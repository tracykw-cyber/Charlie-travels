# My Test Page

A personal website featuring a profile photo and a curated table of dog-friendly places Charlie has visited across Southern California.

## Features

- Profile photo header
- Table of places Charlie has been to, including:
  - Place name and type (hotel, restaurant, beach, etc.)
  - Location
  - Charlie's paw rating
  - Google rating
  - Description / highlights
  - Links to each venue's website

## How to View

Visit the live site at:
wanderwageat.com

Or open `index.html` locally in any web browser.

## Project Structure

```
index.html      # Main webpage
README.md       # This file
```

## Deployment

This site is hosted via **GitHub Pages** with a custom domain.

To deploy updates:
1. Edit `index.html`
2. Upload the updated file to the GitHub repository
3. Changes go live automatically within a few minutes

## Data Source

Place data is sourced from the "Places Charlie Has Been To" Google Sheet, which tracks dog-friendly hotels, restaurants, beaches, and attractions visited.
